El trabajo es la solución a Desafío 2 de JavaScript.
Consiste en tres ejercicios diferentes:
El primero consiste en cambiar el el borde de una imagen.
El segundo consiste en realizar tres input, en los cuales se ingresarán números y devolcerá un párrafo de acuerdo al cumplimiento de una condición.
El tercero es realizar un password por medio de select con núemros de 1 al 9 y verificar si dos números corresponden al password.
La finalidad de éstos tres ejercicios es aplicar las herramientas estudiadas en manipulación del DOM, instrucción If else if y operadores lógicos.
